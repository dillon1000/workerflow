## Suggested reading order

1. [config](./config) and [schema](./schema) for inputs.
2. [connections](./connections) and [auth](./auth) for credentials.
3. [http](./http) plus [effects](./effects) (or [reconcile](./reconcile)) for outbound APIs.
4. [result](./result) and [observe](./observe) for what you return and record.

Concepts: [Idempotent effects](/docs/concepts/idempotent-effects), [Observability](/docs/concepts/observability), [Plugin stdlib](/docs/concepts/plugin-stdlib).
